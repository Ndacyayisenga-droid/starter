Eclipse Starter for Jakarta EE Options not valid for a current selection are disabled. To enable an option, please make the appropriate selection. The following are some examples.
  • To enable Docker support, please select a runtime that supports Docker (amongst the currently included runtimes only GlassFish does not yet support Docker)
  • Only Jakarta EE 10 supports the Core Profile. To enable other Jakarta EE versions, please choose a profile other than Core
  • Jakarta EE 10 does not support Java SE 8. The prior Jakarta EE versions support Java SE 8
  • TomEE and Open Liberty do not yet support Jakarta EE 10, but the other runtimes do
  • GlassFish does not support the Core profile, but it supports the other profiles
